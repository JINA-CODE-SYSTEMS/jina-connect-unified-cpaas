## What is Jina Connect?

Jina Connect is an open-source **Communications Platform as a Service (CPaaS)** that unifies WhatsApp, Telegram, SMS, and RCS into a single, self-hostable Django backend. It's built for teams that need to own their messaging infrastructure without vendor lock-in.

<CardGrid>
  <Card title="Multi-Channel" icon="puzzle">
    WhatsApp (Meta Direct + Gupshup), Telegram, SMS (Twilio, MSG91,
    Fast2SMS), and RCS (Google RBM, Meta RCS) — all through one unified API.
  </Card>
  <Card title="Multi-Tenant" icon="setting">
    Full tenant isolation with RBAC (Owner, Admin, Manager, Agent, Viewer +
    custom roles), per-tenant wallets, and granular permissions.
  </Card>
  <Card title="AI-Native MCP" icon="rocket">
    Built-in Model Context Protocol server — let AI agents send messages,
    manage contacts, and run campaigns through natural language.
  </Card>
  <Card title="Self-Hostable" icon="laptop">
    Docker Compose one-liner or deploy to any VM. PostgreSQL, Redis, Celery —
    all open-source infrastructure. No cloud lock-in.
  </Card>
</CardGrid>
